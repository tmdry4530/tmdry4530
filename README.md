## What I Build

I build agent-assisted tools for local workflows, operator dashboards, trading-signal systems, and product prototypes.

- Maintenance loops that keep existing services healthy.
- Paper-first signal systems with deterministic checks and risk gates.
- Dashboards that show current state, blockers, and the next action.
- TypeScript, React, Node.js, Python, C#, and .NET builds that actually run.
