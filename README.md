> Local agents, workflow dashboards, trading-signal cockpits, and product MVPs with logs, runbooks, and safety gates.

- **News / crypto signal systems** — paper-first, deterministic, risk-gated pipelines
- **Operator consoles** — dashboards that explain state, blockers, and next action
- **Product prototypes** — TypeScript / React / Node / Python / C# builds that can actually run
